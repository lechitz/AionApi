# Quick Diagram Reference

## How to View Diagrams

1) Open `docs/diagram/images/` and click any `.svg`.
2) GitHub will render SVGs directly in the browser.

## How to Edit Diagrams

1) Open the matching `*.sequence.txt` file in `docs/diagram/`.
2) Paste its contents into https://sequencediagram.org/
3) Export as SVG and save into `docs/diagram/images/` with the same base name.
