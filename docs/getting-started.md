# Getting Started

## Requisitos
- Docker e Docker Compose
- Go 1.24+

## Subir ambiente dev
```bash
make dev
make migrate-up   # (opcional) aplica migrations
```

## Rodar testes e cobertura
```bash
make verify
```

## Endpoints
- REST base: `/aion-api`
- GraphQL: `/graphql`
