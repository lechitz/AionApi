## 🔄 Regeneration

To regenerate schema after modifying .graphqls files:

```bash
# Export merged SDL schema
make graphql.schema

# Generate shared queries
make graphql.queries

# Complete setup (schema + queries + docs)
make graphql.setup

# Fetch introspection from running server
make graphql.introspect
```

## 🔧 Integration with Clients

### aion-chat (Python)

Queries are available at `../../graphql/queries/` and can be loaded:

```python
from pathlib import Path

queries_dir = Path(__file__).parent.parent.parent / "AionApi" / "graphql" / "queries"
with open(queries_dir / "categories" / "list.graphql") as f:
    query = f.read()
```

### aionapi-dashboard (TypeScript)

Queries are available at `../AionApi/graphql/queries/`:

```typescript
import fs from 'fs';
import path from 'path';

const queriesDir = path.join(__dirname, '../AionApi/graphql/queries');
const query = fs.readFileSync(
  path.join(queriesDir, 'categories', 'list.graphql'), 
  'utf-8'
);
```
