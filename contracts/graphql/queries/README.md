## 🎯 Purpose

- **DRY Principle:** Define queries once, use everywhere
- **Type Safety:** Ready for GraphQL Codegen integration
- **Versioning:** Queries versioned with schema
- **Consistency:** Same queries in dashboard, aion-chat, and CLI

## 💻 Usage

### Python (aion-chat)
```python
from pathlib import Path

query_path = Path("../AionApi/graphql/queries/categories/list.graphql")
with open(query_path) as f:
    query = f.read()
```

### TypeScript (dashboard)
```typescript
import { readFileSync } from 'fs';
const query = readFileSync('../AionApi/graphql/queries/categories/list.graphql', 'utf-8');
```

## 🔄 Maintenance

Add new query:
```bash
echo 'query MyQuery { ... }' > graphql/queries/domain/operation.graphql
```

Regenerate all:
```bash
make graphql.queries
```
