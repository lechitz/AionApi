# Coverage Artifacts

**Path:** `tests/coverage`

## Overview

Location for test coverage outputs and related reports.
Artifacts here are generated by test/coverage Make targets.

## Typical Workflow

```bash
make test-cover
make test-cover-detail
```

## Design Notes

- Treat files in this folder as generated artifacts.
- Avoid manual edits to generated report files.
- Keep report generation reproducible via Make targets.

## Package Improvements

- Add artifact retention policy for local/CI usage.
- Add quick command matrix for each report type.
- Add ignore/cleanup guidance for large report files.
- Add pointers to CI coverage publication path.
